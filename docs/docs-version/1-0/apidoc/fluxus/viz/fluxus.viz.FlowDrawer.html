fluxus.viz.FlowDrawer#

class fluxus.viz.FlowDrawer(style=None)[source]#

A drawer for flows.

Available styles:

  • FlowTextStyle or “text”

  • FlowGraphStyle or one of “graph”, “graph_dark”
