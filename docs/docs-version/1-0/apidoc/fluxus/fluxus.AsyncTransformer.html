fluxus.AsyncTransformer#

class fluxus.AsyncTransformer[source]#

A transformer designed for asynchronous I/O.

Synchronous iteration is supported but discouraged, as it creates a new event loop and blocks the current thread until the iteration is complete. It is preferable to use asynchronous iteration instead.

final transform(source_product)[source]#

Generate a new product, using an existing product as input.

This method is implemented for compatibility with synchronous code, but preferably, atransform() should be used instead and called from within an event loop.

When called from outside an event loop, this method will create an event loop using arun(), transform the product using atransform() and block the current thread until the iteration is complete. The new product will then be returned.

Parameters:

source_product (AsyncTransformer) – the existing product to use as input

Return type:
