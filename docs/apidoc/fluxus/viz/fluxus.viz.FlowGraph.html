fluxus.viz.FlowGraph#

class fluxus.viz.FlowGraph(*, single_conduits=None, connections=None)[source]#

A graph representation of a Flow, or a partial flow represented by any type of Conduit.

Parameters:

  • single_conduits (Optional[set[SerialConduit[Any]]]) – the set of all single, unconnected conduits in the graph (optional)

  • connections (Optional[set[tuple[SerialConduit[Any], SerialConduit[Any]]]]) – the set of all connections between conduits in the graph (optional)

to_dot(*, width=7.5, font=None, fontsize=None, fontcolor=None, fontcolor_terminal=None, background=None, foreground=None, fill=None, stroke=None)[source]#

Convert this flow graph to a DOT graph representation.

Parameters:

  • width (Optional[float]) – the width of the graph in inches, or None for unconstrained width (defaults to 7.5 inches, fitting letter or A4 pages in portrait mode)

  • font (Optional[str]) – the font of the graph (optional)

  • fontsize (Optional[float]) – the font size (optional)

  • fontcolor (Union[RgbColor, RgbaColor, None]) – the text color of the graph (defaults to the foreground color)

  • fontcolor_terminal (Union[RgbColor, RgbaColor, None]) – the text color of the terminal nodes (defaults to the font color of regular nodes)

  • background (Union[RgbColor, RgbaColor, None]) – the background color of the graph (optional)

  • foreground (Union[RgbColor, RgbaColor, None]) – the foreground color of the graph (optional)

  • fill (Union[RgbColor, RgbaColor, None]) – the fill color of the nodes (optional)

  • stroke (Union[RgbColor, RgbaColor, None]) – the stroke color of the nodes and edges (optional)

Return type:

str

the DOT graph representation
