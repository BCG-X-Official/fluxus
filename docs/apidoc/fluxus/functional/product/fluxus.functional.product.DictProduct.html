fluxus.functional.product.DictProduct#

class fluxus.functional.product.DictProduct(name, product_attributes, *, precursor=None, start_time, end_time)[source]#

A flow product that consists of an attribute-value mapping.

The product name is the name of the step that created this product.

The attribute names must be valid Python identifiers.

Bases:

HasLineage [DictProduct | None], HasDictRepr, Mapping [str, Any]

Metaclasses:

ABCMeta

Parameters:

  • name (str) – the name of this product

  • product_attributes (Mapping[str, Any]) – the attributes that were changed or added by the step that created this product, comprising both fixed attributes and dynamically generated attributes

  • precursor (Optional[DictProduct]) – the precursor of this product (optional)

  • start_time (float) – the start CPU time of the step that created this product, in seconds since an arbitrary starting point

  • end_time (float) – the end CPU time of the step that created this product, in seconds since an arbitrary starting point

Raises:

  • TypeError – if the product attributes are not a mapping

  • ValueError – if the attribute names are not valid Python identifiers

get_lineage_attributes()#

Get the attributes and labels of this product’s lineage.

Creates a nested dictionary where the first level keys are product names and the second level keys are attribute/label names. The second level values are attribute/label values.

Product names are unique names for each product in the lineage. This is the class name of the product, followed by an ascending integer if the same class appears multiple times in the lineage. For example, if the lineage is [A(…), B(…), A(…), A(…), C(…), B(…)], the product names will be ["A", "B", "A#1", "A#2", "C", "B#1"].

If a label has the same name as an attribute, the label’s value will override the attribute’s value in the dictionary. For example, if a product A(…) has an attribute "name" and a label "name", the label’s value will be used for the key {"A": {"name": …}}.

Return type:

dict[str, dict[str, Any]]

Returns:

the attributes and labels of the lineage as a nested dictionary

label(**labels)#

Label this product with the given labels and return self.

This means that labelling can be easily inserted into the code anywhere the product is used, e.g., after object creation, replacing

product = Product()

with

product = Product().label(label1=value1).label(label2=value2)

or, even better

product = Product().label(label1=value1, label2=value2)

If the labels are already stored in a dictionary, they can be passed as keyword arguments using the ** operator:

labels = {"label1": value1, "label2": value2}
product = Product().label(**labels)
